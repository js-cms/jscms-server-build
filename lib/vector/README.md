# Jscms

欢迎使用Jscms，请访问以下链接获取帮助。

- 官方网站：[jscms.top](http://www.jscms.top/)。
- GitHub地址：[https://github.com/js-cms](https://github.com/js-cms)。

## 简单帮助

当你使用jscms-cli初始化完毕后，你应该对项目进行依赖安装：

```bash
npm install
```

安装完毕依赖之后，运行以下命令启动项目

```bash
npm start
```

停止项目的命令如下：

```bash
npm stop
```
